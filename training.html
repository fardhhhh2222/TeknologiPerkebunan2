Bootcamp Smart Farming: IoT untuk Perkebunan 4.0

Curriculum

Hari 1: Dasar-dasar IoT

Pengenalan konsep IoT, komponen, dan aplikasinya dalam perkebunan.

Hari 2: Sensor dan Perangkat

Berbagai jenis sensor untuk monitoring tanah, air, dan kondisi cuaca.

Hari 3: Pengumpulan dan Analisa Data

Setup sistem pengumpulan data dan platform analitik untuk perkebunan.

Hari 4: Otomatisasi Perkebunan

Implementasi sistem otomatisasi irigasi dan pemupukan berbasis IoT.

Hari 5: Proyek Implementasi

Praktik langsung membuat mini-project IoT untuk perkebunan.
